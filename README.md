# ポートフォリオサイト 編集ガイド

## 📁 構成

```
portfolio/
├── index.html    ← このファイル1つで完結
└── images/       ← 作品画像を入れるフォルダ（自分で作る）
```

---

## ✏️ 編集する場所（全て `index.html` 内）

### 1. 名前・プロフィールを書き換える

`index.html` を開いて、以下を検索・置換してください。

| 探す文字列 | 置き換える内容 |
|---|---|
| `R. Tachibana` | 英字の名前（3箇所） |
| `your@email.com` | 実際のメールアドレス |
| `[ここに経歴・実績を書いてください。...]` | 経歴を記入 |
| `[制作で大切にしていることを...]` | 制作姿勢を記入 |

Socialリンク（Instagram / Behance / Vimeo）も `href="#"` を実際のURLに置き換えてください。

---

### 2. 作品を追加・編集する

`index.html` の下の方、`// 📝 作品データ` と書かれたブロックを編集します。

#### 🎬 動画作品（上段）

```javascript
{ title: '作品名', year: '2025', youtubeId: 'dQw4w9WgXcQ' }
```

- **youtubeId の取り方**: YouTubeの動画URLが `https://www.youtube.com/watch?v=dQw4w9WgXcQ` なら、`dQw4w9WgXcQ` の部分（11文字）をコピー
- サムネイル画像は自動で取得されます
- クリックすると画面中央で再生（ライトボックス）

#### 🖼️ グラフィック作品（下段）

```javascript
{ title: '作品名', year: '2025', image: 'images/work01.jpg' }
```

- 画像は `images/` フォルダに入れる
- 推奨: JPEG/WebP、縦長（3:4比率が最適）、1枚 500KB 以下に圧縮
- クリックすると原寸大で拡大表示

---

### 3. 作品を削除する

対象の行ごと削除するだけ。（末尾のカンマにご注意）

## 💡 見た目の調整

### スクロールの速度を変えたい
`index.html` の中で以下の部分を探してください:

```html
<div class="marquee" style="--duration: 80s;">   ← 動画の流れる速度
<div class="marquee reverse" style="--duration: 110s;">  ← 画像の流れる速度
```
- 数字を**大きく**すると**遅く**なります（例: 120s = 120秒で1周）
- 数字を**小さく**すると**速く**なります

### 色を変えたい
`:root { ... }` の中の `--ink` などを変更:
```css
--bg: #ffffff;    /* 背景色 */
--ink: #1a1a1a;   /* 文字色 */
--ink-soft: #888; /* 薄い文字色 */
```

## 🌐 公開する（Vercel、10分で完了）

1. **GitHubアカウント作成**: https://github.com
2. 新規リポジトリ作成 → `index.html` と `images/` フォルダをアップロード
3. **Vercelアカウント作成**: https://vercel.com （GitHubでログイン可）
4. 「New Project」→ リポジトリを選択 → Deploy
5. `your-name.vercel.app` のようなURLが発行される

## ⚠️ 重要な注意事項

### YouTube動画について
- **動画は限定公開でもOKです**（「非公開」は不可）。URLを知っている人だけが見られる「限定公開」設定を推奨
- サムネイル画質は動画投稿時に「`maxresdefault`」相当の高画質が自動生成されます。古い動画だと低画質になるので、必要に応じて**YouTube Studioでカスタムサムネイルを設定**してください

### 画像の最適化
- **必ず圧縮してからアップロード**してください（重いと離脱されます）
- 推奨ツール: [Squoosh](https://squoosh.app/) または [TinyPNG](https://tinypng.com/)
- 目安: 1枚あたり 200〜500KB

### 作品数の目安
- 現在のサンプル: Motion 8点 / Graphic 14点
- **増やしすぎると流れが長くなりすぎる**ため、20〜30点程度を目安に厳選してください

## 🎯 クリエイティブ業界での見せ方のコツ

1. **最初の3点が勝負** — 横スクロールの最初に表示される作品が第一印象を決めます
2. **Motion と Graphic のバランス** — どちらも同じくらい流すと「両方できる人」が伝わりやすい
3. **動画のサムネイルは統一感を意識** — トーンがバラバラだと流れが雑然として見えます
4. **Aboutの経歴は時系列より「得意領域」から書く** — 採用担当は最初の1〜2行で判断します
